Техническое задание (ТЗ)

ПроектResourcePlanSymphony

1. Общая информация

1.1 Цель проекта

Разработать веб-систему планирования ресурсов команд на базе графиков отпусков сотрудников с формированием итогового Excel-отчёта.

1.2 Целевая платформа

  • Веб-приложение для внутреннего использования.
  • Поддержка современных браузеров (Chrome, Edge).

1.3 Технологический стек

  • PHP 8.3+, Symfony 7.x
  • PostgreSQL 17
  • Symfony UX + Twig + JavaScript
  • Excel: PHPSpreadsheet

2. Область работ

2.1 Включено в объем

  • Загрузка и парсинг Excel-файла отпусков.
  • Настройка ролей, команд и распределения времени.
  • Выбор состава команды и периода планирования.
  • Расчёт доступности по неделям и ролям.
  • Экспорт результата в Excel-шаблон.
  • Хранение состояния в PostgreSQL с fallback на файл.

2.2 Не включено

  • Полная аутентификация и RBAC.
  • Интеграции с HRM/ERP сверх календарного API.
  • Многоязычность.
