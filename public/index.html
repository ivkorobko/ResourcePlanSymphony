Ресурсный план за несколько шагов

Файл отпусков загружается в исходном формате, сотрудники группируются по подразделениям, а итоговая выгрузка собирается на базе шаблона ресурсного плана.

1

Загрузка файла

Загрузите файл графика отпусков в исходном Excel-формате.

Ожидание
2

Сохранение данных

Выберите команду из настроек. После выбора сохранённый состав команды будет подставлен в подразделения и сотрудников.

Ожидание
3

Подразделения

Выберите одно или несколько подразделений для планирования.

Ожидание
4

Сотрудники

Отметьте сотрудников внутри выбранных подразделений. Должность для каждого берётся из загруженного файла.

Ожидание
5

Период

Выберите подряд идущие месяцы. Для шаблонного экспорта доступно до трёх месяцев.

Ожидание
6

Анализ

Проверьте состав, выбранный период и недельную нагрузку перед выгрузкой.

Ожидание
7

Ресурсный план

Скачайте Excel-файл по шаблону со сводным и детальным планом.

Ожидание

Подключение календаря

Параметры внешнего API для расчёта рабочих дней и праздничных исключений.
